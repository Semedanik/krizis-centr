## Шаги для первого коммита и пуша:

### 1. Добавьте файлы в staging area:
```bash
git add .
```

### 2. Создайте первый коммит:
```bash
git commit -m "Initial commit: Кризисный центр помощи женщинам и детям"
```

### 3. Создайте репозиторий на GitHub:
- Зайдите на https://github.com
- Создайте новый репозиторий (New Repository)
- **НЕ** добавляйте README, .gitignore или лицензию (они уже есть)

### 4. Подключите удаленный репозиторий:
```bash
git remote add origin https://github.com/ВАШ_USERNAME/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
```

### 5. Переименуйте ветку в main (если нужно):
```bash
git branch -M main
```

### 6. Запушьте код на GitHub:
```bash
git push -u origin main
```
